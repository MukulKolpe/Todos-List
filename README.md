# Todos-List

## Description
It is a simple Todos-List made using React.js
## Tech-stack Used
- React.js
- CSS
- JavaScript

## Prerequisites

Make sure you have node.js installed.

### How to check?
 ```
 node --version
 ```
 ```
 npm --version
 ```
 ```
 npx --version
 ```

## Setting up locally
1. Fork this repository.
2. clone it.
 ```
 git clone https://github.com/YOUR_GITHUB_UserName/Todos-List
 ```
3. 
 ```
 cd Todos-List
 ```
4. Install node modules
 ```
 npm install
 ```
5. 
 ```
 npm start
 ```
